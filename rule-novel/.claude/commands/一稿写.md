# /一稿:写 <范围>

一稿快速铺量。**Claude是作者，Agent是工具。**

## 每批流程

### 第0步：创作意图（Claude自己做）
- 读章纲 + 前3章正文 + 钩子追踪 + 代价账本
- **自己判断**：这5章的情绪弧线？节奏呼吸？哪章读者会停下来回读？
- 形成创作意图——不是Agent生成的，是我的

### 第1步：逐章起草
- 把意图+章纲喂给 chapter-writer Agent
- Agent写完 → **我读一遍**。章纲全中但读起来不对 → 我改或重写
- Agent服从我的判断。不是反过来
- 我觉得"可以了" → 才进入验证

### 第2步：Agent验证（参考）
- chapter-validator + cost-ledger(快速) + hook-tracker 并行
- Agent标记是**参考信息**——不是命令
- 我逐条判断：修/不修/记到二稿

### 第3步：批内连读
- 5章写完我通读一遍——连起来感受
- 节奏对吗？有没有两章太像？代价线断了吗？
