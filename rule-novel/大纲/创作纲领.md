# 创作纲领 (Brief)

> **版本**：v2.0 — 烙印体系
> **最后更新**：2026-07-03

## 基本信息
- **书名**：《请确认您已阅读并同意本规则》
- **题材/类型**：规则怪谈 / 悬疑 / 都市
- **目标总字数**：100-150 万字
- **发布平台**：番茄小说
- **目标读者**：男频，18-35 岁，喜欢烧脑悬疑和反套路

## 一句话梗概
> 规则怪谈降临的世界里，一个普通人发现自己能用自己的记忆为代价篡改规则——但他不记得自己已经改过多少条，也不记得自己忘了多少人。而每一次遗忘，都在他身上留下一道烙印——那是代价的残骸，也是对抗规则系统的唯一武器。

## 核心卖点（4 个）

1. **规则可以被改——但不是免费的。** 每改一条规则，就失去一段记忆。代价从恐惧中生成——你越怕什么，什么越被拿走。

2. **代价不只是支出——它在你身上留下烙印。** 你忘了奶奶的脸——但奶奶的脸在你的手指上变成了一道永远不会愈合的伤口。那道伤口能驳回规则。代价不是纯粹的失去——是转化。你失去的东西换了形态，留在你身上，成为你的武器。

3. **烙印之间会共振——产生全新的能力。** 两个烙印单独使用只是豁免——但放在一起，它们会产生第三种效果。不是1+1=2，是1+1=一个新东西。读者追的是："下一个烙印会解锁什么共振？"

4. **反套路开篇。** 不写系统觉醒，不写金手指降临。第一幕从陈渡的日常切入——重复的电话、不记得的承诺、身体上无法愈合的伤口。异常不是从天而降的，是渗透进来的。读者和陈渡一起慢慢发现：他的遗忘不是偶然，他的伤口不是意外，他周围的日常已经被某种东西穿过了。渐进式渗透——用冷感笔法建立不安，以差异化竞争。前500字无规则要素，但必须让读者感到"有什么东西不对"。

## 主角概要

- **姓名**：陈渡
- **年龄**：26
- **身份**：前网文扑街作者，现规则对策局文职人员（档案管理岗）
- **核心欲望**：找回自己失去的记忆，搞清楚自己到底改过多少规则，弄明白身上的烙印到底能做什么
- **内在缺陷**：他不敢面对自己改规则后失去的记忆——那些他忘掉的人，可能很重要。他也不敢使用烙印——因为每次使用，伤口就又深一点
- **能力体系**：
  - **改写**：用血在规则纸上书写，覆盖/追加规则
  - **收容**：改写场域后，可将场域背后的规则之灵收容
  - **烙印**：每次支付代价后，身体上留下一个烙印。烙印有基础能力，且之间可以共振产生新能力
  - **代价**：每次使用能力=支付代价=失去记忆/感官/功能。每次使用烙印=支付利息=代价持续加深
- **成长弧线**：从隐藏烙印 → 发现烙印能做什么 → 解锁烙印共振 → 收集更强烙印 → 面对烙印的最终代价

## 力量体系概要

### 规则之灵（敌人 = 力量来源）
- 每一个规则场域背后有一个"规则之灵"——书写者碎裂后的意识碎片
- 灵体等级：残灵（D级）→ 碎灵（C级，红风衣形态）→ 整灵（B级，形态各异）→ 大灵（A级）→ 元灵碎片（S级）
- 每个灵体有一条不可违背的"核心规律"

### 改写者（主角的类型）
- 用血在规则纸上写 = 支付代价 = 获得改写权限
- 改写后可收容灵体 = 获得烙印
- 烙印有基础能力（对规则的"豁免权"）
- 多个烙印之间共振 = 产生全新能力

### 烙印（代价转化成的武器）
- 代价值的烙印等级高：印记→刻印→烙印→圣痕→元痕
- 烙印使用需支付利息（代价的持续吞噬）
- 共振发现是核心爽点——里程碑式能力解锁

## 世界观概要

- **规则降临**：2024年3月开始，全球出现"规则场域"——进入者必须遵守规则，违反即死
- **规则来源**：书写者——最初的一个人类改写者，碎裂后变成无数规则之灵
- **一级书写者**：数字为零——被一个叫R-00的女孩用自己的存在为代价删除了
- **规策局**：死亡率47%的官方机构。军方在内部安插了"归档计划"——秘密追捕能力者
- **能力者生态**：五类（改写/读取/免疫/共鸣/锚定），改写者最稀有且是唯一能收容灵体的

## 创意约束

- **反套路设计**：不是"获得系统变强"，是"已经变强了但不记得代价"。烙印不是奖励——是伤口
- **硬约束1**：代价不可逆——不能通过"写一条'我恢复所有记忆'"来解决问题
- **硬约束2**：烙印不可转移——但军方在研究如何突破（这是第三卷的核心推动力）
- **硬约束3**：烙印使用 = 代价加深。没有免费的胜利。爽感来自"我用伤口赢了你"而不是"我有超能力"
- **对标作品**：《神秘复苏》（鬼的驾驭体系）、《十日终焉》（规则博弈）、《诡舍》（规则生存）

## 追读力设计

- **短期钩子**（每章）：代价的利息扣了什么？伤口蔓延到哪了？日记空白又扩大了吗？
- **中期钩子**（每5-10章）：下一个烙印是什么？烙印之间会发生共振吗？
- **长期钩子**（每卷）：书写者是谁？一级为什么是零？代价收走的记忆去了哪？
