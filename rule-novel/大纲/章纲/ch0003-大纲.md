# 第三章 长安北路

## 章纲（已写，反向记录）

### 节拍
1. **现场探察**：陈渡到长安北路十字路口——老周派他来提取后台日志。路口没有纸，但左手旧疤在发凉。发现路面沥青有一块颜色不对——规则纸的残留投影。右手食指在接近时渗血，两个烙印同时振动、互相通信。
2. **数据分析**：第二天，陈渡把分析仪后台日志和老数据比对——发现异常低频（不到1Hz），是纸本身的频率。和地下四层周小禾的纸频率一致。老周打断他——"一样。"两人意识到：长安北路的纸被归档计划截走了。
3. **赵明远预告**：老周告知赵明远明天会来审问陈渡——带录音笔。赵明远已经知道右手伤口不是意外，正在逐步缩小搜索范围。老周说——在回答赵明远之前，先跟他去地下四层。
4. **地下四层**：货梯只到地下三层——地下四层按钮被物理移除。消防楼梯往下，防火门，走廊两侧铁门P-1到P-17。最里面一间——"东郊D级场域残留——当事人：周小禾"。老周在门前停了很久。
5. **周小禾的规则纸**：一张看起来全新的A4纸——"您已违反本条规则。"陈渡用食指血触碰——纸面文字褪色，背面浮出隐藏文字："一级书写者人数：零。"下面手写两个字："爸爸。"
6. **赵明远审问**：第二天，赵明远准时到。问"九月十七日你在做什么"。陈渡说真话——老周教他的。赵明远走到门口停住——"你知道你左手那道疤不是意外。那是代价。"然后透露：有人在保护陈渡，但军方快到了。

### 本章核心信息
- 烙印共振首次展示——两个烙印在不同频率上互相通信，不是陈渡主动触发
- 规则纸残留——纸被移走后投影还在，烙印能感应
- 地下四层物品间——规策局最高机密区域
- "一级书写者人数：零"——全书核心谜题首次揭露
- 周小禾的最后两个字"爸爸"——情感重音
- 赵明远身份模糊化——不是单纯追查者，说"那不是意外。那是代价。"显示他知道更多

### 本章基调
- 前半段：冷感探察——长安北路现场的物理细节
- 中段：悬疑推进——数据分析→地下四层潜入
- 后半段：情感揭示（周小禾→爸爸）+ 审问压力
- 章末：赵明远的"有人在保护你"——留下开放悬念
