# 人物形象弧度蒸馏器

> 用途：提炼大火网文的人物塑造规律。服务于角色创建、弧线设计、配角系统。不止一本小说。

---

## 参考样本

### 《异兽迷城》高阳——三段式成长弧

| 阶段 | 身份 | 核心驱动 | 读者情绪 |
|------|------|----------|----------|
| 第一阶段 | 普通人/幸存者 | 觉醒【幸运】，被动求生，会恐惧会迷茫 | 心疼+代入 |
| 第二阶段 | 觉醒者/组织成员 | 加入十二生肖→麒麟公会七影长老，主动出击 | 跟随+期待 |
| 第三阶段 | 九嗣队长/守护者 | 格局从"小我"跃升至"大我"，拯救人类 | 燃+泪 |

**关键手法**：每次升级都有代价。不是"升级了更强了"——是"失去了一样东西，换来了成长"。代价=成长的可信度。

### 《十日终焉》齐夏——反套路坚定型

主角从头到尾不变核心价值观。他不被世界改变——他改变世界。弧线不在"他从A变成B"，而在"读者从以为他是A → 逐渐发现他其实是C"。信息弧线替代了性格弧线。

### 《斩神》林七夜——能力层级递进

成长不是线性升级——是跳跃的。每次能力跃迁都绑定一个人际关系转折。能力+关系双线交织。

## 五条通用规律

### 1. 主角弧线 = 缺陷 → 催化剂 → 三次渐进转变 → 代价闭环

主角不能一开始就完整。完整=没有成长空间。

| 弧线节点 | 位置 | 功能 | 禁止 |
|----------|------|------|------|
| 初始缺陷 | 前5章 | 建立"内在谎言"——主角相信的错误认知 | 不能是外在缺陷（穷/弱），必须是内在的（不信人/不敢要/觉得自己不配） |
| 催化剂 | 第3-5章 | 直击缺陷的事件——逼主角面对 | 不能是旁白告知，必须是一个场景 |
| 转变节点1 | ~25% | 面对缺陷——不适——第一次尝试改变 | 不能一次成功 |
| 转变节点2 | ~50% | 旧模式崩溃——主动选择新方式——代价支付 | 必须付代价 |
| 转变节点3 | ~75% | 新方式稳固——外部验证——但发现新问题 | 不能完全解决 |
| 代价闭环 | 结局 | 回看起点——已不是同一个人——但带着伤 | 代价不能消失 |

**本作陈渡的弧线校验**：

| 节点 | 当前设计 | 
|------|----------|
| 初始缺陷 | "习惯了"——九年等待，麻木的日常，不敢走出去 |
| 催化剂 | 妹妹留言"哥。我找到门了" |
| 转变节点1 | 出走廊？书店？ |
| 转变节点2 | 四滴完整？面对书写者？ |
| 转变节点3 | 代价从"被迫扣"变"版税"？ |
| 代价闭环 | ch15 "你们好"——代价=合作出版 |

### 2. 配角五型——每一种只承担一个功能

五类配角，每类只做一件事，但要把那件事做到让读者记住：

| 类型 | 功能 | 设计要点 | 反面教材 |
|------|------|----------|----------|
| **死对头** | 拉高格局 | 与主角势均力敌的成长速度 + 不为人知的弱点 | 扁平反派（我就是坏） |
| **死党** | 衬托品质 | 与主角相反的性格 + 经常拖累主角 + 关键时刻出生入死 | 工具人跟班（只会说666） |
| **导师** | 给予方向 | 自身有冲突有秘密 + 连接主线 + 关系可演变 | 纯信息输出机 |
| **竞争者** | 催化进步 | 亦敌亦友 + 你追我赶 + 理念分歧 | 纯垫脚石 |
| **情感坐标** | 保持人性 | 主角的软肋 + 行为逻辑的锚点 + 不能只是"要被救的公主" | 花瓶/背景板 |

**本作配角校验**：

| 角色 | 类型 | 功能性是否清晰？ |
|------|------|------------------|
| 陈屿 | 情感坐标 | ✅ 妹妹=代价的动机，密码=九年的锚点 |
| 妈妈 | 情感坐标+导师 | ✅ 走廊里的等待，"门"的来源 |
| 奶奶 | 导师(已故) | ✅ 硬抄本+折纸+四滴传承 |
| 苏姨 | 导师+情感坐标 | ✅ 书店=安全屋+无字书+奶奶日记 |
| 顾伯 | 导师+竞争者 | ✅ 名册+原始纸+背面的人 |
| 林远 | 竞争者+死党 | ✅ 公式+修/拆分歧+馒头泡面 |
| 程 | 死党(远程) | ✅ 啦啦队+硬抄本留言 |
| 书写者 | 死对头→竞争者→？ | 弧线最复杂——需要追踪 |

### 3. 配角记忆点 = 一个具体物 + 一个固定动作 + 一个反差

配角的出场字数有限。用三个要素建立辨识度：

| 要素 | 示例 | 
|------|------|
| 具体物 | 林远的馒头+榨菜塑料袋 / 奶奶的硬抄本 / 苏姨的收音机 |
| 固定动作 | 林远在空气里推公式 / 苏姨摸无字书 / 顾伯身体正反面闪 |
| 反差 | 林远用公式拆规则但买馒头忘了泡面 / 苏姨眼花了但闭着眼找书 |

**一句话法则**：读者合上书一周后还能凭一个字想起一个配角 = 配角的记忆点到位了。

### 4. 人物关系网的驱动原则

每个人物之间的关系不是"认识/不认识"——是**在同一个问题上的立场不同**。

| 关系类型 | 驱动方式 | 示例 |
|----------|----------|------|
| 同一问题·同一立场 | 结盟/信任/温暖 | 苏姨+陈渡——都在找"被涂掉的人" |
| 同一问题·不同方法 | 张力/争论/互相推动 | 林远+陈渡——拆派vs修派 |
| 同一问题·不同信息 | 揭示/颠覆/重新理解 | 程=入——"入"是倒过来的"人" |
| 不同问题·互相影响 | 意外/扩容/世界观打开 | 顾伯+陈渡——规则vs纸外 |

**禁止**：两个角色的关系只靠"他们认识/是朋友/是敌人"来定义。必须有一个**具体的问题**把他们的立场分成两个位置。

### 5. 读者对角色的情感投资曲线

读者对一个角色的感情不是一次性投入的——是分阶段累积的：

| 阶段 | 读者心理 | 需要的东西 |
|------|----------|-----------|
| 第1-3章 | "这人有点意思" | 一个具体的习惯/缺陷/执念 |
| 第4-10章 | "我希望他赢" | 他为一个具体的人/东西付出了代价 |
| 第11-20章 | "我怕他输" | 他在乎的东西受到威胁 |
| 第21-40章 | "我跟他一起走到了这里" | 他做过一次让读者意外的选择（不是剧情需要——是他自己选的） |

**公式**：认识→关心→害怕失去→共同经历→意外选择 = 完整的读者情感投资。

## 通用模板：角色设计自检表

```
主角：
□ 内在缺陷是什么？（不是穷/弱，是信错了东西）
□ 催化剂在第几章？（必须是一个场景，不是旁白）
□ 三次转变节点分别在哪里？
□ 代价闭环——结局时他失去了什么，换来什么？

配角：
□ 每个配角只承担一个功能（五型选一）
□ 每个配角有一个具体物+固定动作+反差
□ 每个配角和一个具体问题的立场绑定
□ 至少有一个配角在关键时刻做过让读者意外的选择

关系网：
□ 每对核心关系有一个"具体问题"驱动
□ 不是"他们是朋友/敌人"——是"他们在某问题上各自站在哪边"
```

## 本作特化：规则怪谈的人物额外约束

规则怪谈的文类特性给人物多了一层要求：

| 约束 | 说明 |
|------|------|
| 规则不能只是设定 | 规则必须通过人物的身体感受来呈现——代价的白线、伤口的方向、疼=排斥 |
| 代价必须可见 | 不能只说"他付出了代价"——必须有一个可以摸/看/数/量的东西在变 |
| 被涂掉的人必须有质感 | 陈屿不能只是一个"被涂掉"的概念——她有歪的"门"字、缺的门牙、甩过来的马尾 |
| 人物不能在规则面前变小 | 规则系统很庞大——但人物不能变成它的附庸。人>规则 |
```

**重要**：这是第一次写——后面根据我们的实际经验迭代修正。
