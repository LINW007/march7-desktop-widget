# 网文开篇蒸馏器 · 规则怪谈向

> 用途：提炼大火网文的通用开篇规律，校准本作开篇。不止服务一本小说——是方法论。

## 参考样本

### 《异兽迷城》——"平地一声雷"

| 段落 | 内容 | 字数占比 | 功能 |
|------|------|----------|------|
| 日常平地 | 高阳穿越12年，温馨家庭早餐——父母、奶奶、妹妹 | ~500字 (约25%) | 读者投射主角，建立情感依恋 |
| 导火索 | 巷子被撞，中年人喊"到处都是怪物"→被警察当场击毙 | ~400字 (约20%) | 一记炸雷——日常碎了 |
| 涟漪 | 回忆爷爷房间怪声、手变鳞片、离奇死亡 | ~500字 (约25%) | 信息分批释放，涟漪扩散 |
| 梦中炸裂 | 尸体坐起竟是那个中年人 | ~200字 (约10%) | 第二记雷——恐惧升级 |
| 章末钩子 | 微信消息——喷出一口水 | ~400字 (约20%) | 标准cliffhanger——读者必翻页 |

**核心法则**：先建后破。平地越温馨、越平淡、越美好——雷就炸得越响亮。

> 鲁迅："悲剧就是把美好的东西撕碎给人看。"

## 五条通用规律

### 1. 日常平地的情感重量

"日常"不是空白——是让读者在异常闯入**之前**就建立情感投射。

| 做法 | 效果 |
|------|------|
| 写角色在做什么（动作/习惯/身体状态） | 读者认识"他是谁" |
| 写角色在乎什么（具体的物/人/记忆） | 读者建立情感联结 |
| 写角色的失去/缺憾（淡淡的，不外露） | 读者心里开始痛 |
| **不要**写角色在想什么/这个世界怎么回事 | 那是第二章的事 |

**关键**：日常平地里不能"空白"。要以**情感重量**为核心——读者必须对至少一个具体的人/物/关系产生好感或同情。

### 2. 异常至少分两波

第一波异常（导火索）——打破日常，但不揭示全部真相。
第二波异常（升级）——在读者以为"原来如此"之后再加一记。
第三波才揭示能力/系统（如果有）。

| 波次 | 功能 | 读者状态 |
|------|------|----------|
| 第一波 | 刺破日常——外面有什么不对 | 警觉 |
| 第二波 | 升级——不只是不对，是危险/是机会 | 紧张 |
| 第三波 | 揭示能力/真相——这个不对有解释 | "原来如此"→然后更多疑问 |

**禁止**：第一波和第二波之间插入大段设定叙述。

### 3. 用具体物锚定抽象概念

不说"这个世界有规则"——说"纸折四下。打开。背面浮出了字。"

| 抽象 → 具体转换 |
|------------------|
| "规则系统控制世界" → "重力是'原生'的。时间是'原生'的。" |
| "妹妹被抹去了存在" → "她在所有人眼里——只是空气。" |
| "代价在吞噬他" → "白线经过的地方——汗毛没了，毛孔在闭合。" |

**法则**：每一段至少出现一个可以摸/看/闻的东西。

### 4. 信息释放节奏：涟漪式

| 时间轴 | 信息量 | 读者 |
|--------|--------|------|
| 开篇 | 1个异常事件 + 1个人物锚点 | "嗯？这怎么回事？" |
| 中段 | 2-3个涟漪信息（回忆/他者叙述/物品变化） | "等等……所以是……？" |
| 章末 | 1个颠覆性信息 + 1个钩子 | "然后呢？！" |

**禁止**：在同一段或相邻两段里放超过2个新信息。

### 5. 章末必须是一个"动作"

不是"他明白了"——是"他把右手食指压在那行字上"。不是"新的旅程开始了"——是"门无声无息地开了一条缝"。

|------|-----------|------|
| 日常平地 | 第1段就是能力（折纸浮字） | 读者没建立"他是谁"就撞进异常 |

```
v3 结构：

[第一幕] 日常平地（~400字）
→ 陈渡下班。出租屋。十七平米。窗外的长安北路。
→ 桌上：硬抄本、墨汁、妹妹的照片。不是照片——是奶奶画的铅笔素描。因为照片里也拍不到她。
→ 九年。他习惯了。每天做的事：折纸、看规则、记硬抄本。
→ 右手食指上的口子不会愈合。痂的边缘有一圈白——爬过了第三指节。
→ 安静的日常。机械的日常。九年的日常。

[第二幕] 第一波异常——门锁（~300字）
→ 凌晨三点零六分。他准备关灯。
→ 门锁。再等一分钟。每天凌晨三点零七分——锁舌自动弹开。
→ 他从来没走出去过。九年了。
→ 三点零七分。锁舌弹开。门无声无息地开了一条缝。
→ 他把门推上。像每天一样。但今天——门板合上之后——那条缝比平时宽了一点点。

[第三幕] 第二波异常——玻璃里的脸（~300字）
→ 玻璃里的倒影。
→ 他自己的脸——后面还有一张脸。很小。很淡。扎马尾。
→ 九年了——她第一次出现在玻璃里。
→ 她张开嘴。无声地说了三个字。
→ "找到了。"

[第四幕] 能力揭示——规则纸+妹妹留言（~600字）
→ 陈渡回到桌前。把那张折了四下的纸翻到背面。
→ 二十年的习惯：扫一眼底层的规则签名。
→ 但他停住了。
→ 在所有规则的下面。压着一行字——不是浮出来的，刻在纤维里的。
→ "哥。我找到门了。……"
→ 活的。她还活着。

[第五幕] 行动+钩子（~400字）
→ 陈渡翻开硬抄本。写好前三条规则。
→ 右手食指还在渗。他碰了一下纸上的那行字。
→ 伤口不痒了。三年来的第一次。
→ 纸的另一面——有人在用指甲——从底下——轻轻地——顶了回来。
```

## 使用指南

以后写任何新书开篇，按这个流程自查：

1. **日常平地有没有情感重量？** 读者能不能在异常闯入之前对主角产生共情？
2. **异常是否至少两波？** 刺→加深→揭示——不是一次性砸完。
3. **有没有用具体物锚定抽象？** 每段至少一个可触/可嗅/可听的锚点。
4. **信息是不是涟漪式释放？** 同一段不超过2个新信息。
5. **章末是不是一个动作？** 不是总结，不是感想——是一个正在发生的动作。
