# 对标蒸馏 · 三书

> 十日终焉 · 异兽迷城 · 神秘复苏。各取一根骨头。装进我们的骨架。

---

## 一、《十日终焉》→ 取"过肩视角 + 规则驱动"

### 为什么爆

- 番茄榜首，200万人同时追更，中国作协重点扶持，中国小说学会2024年度好小说第一名
- 类型融合：无限流 × 规则文。所有关卡完全原创，不依赖现成IP
- 作者杀虫队队员自述核心：**"摄像机架在主角肩膀上"** 的跟随视角

### 我们取什么

| 技法 | 说明 | 本作落地 |
|------|------|----------|
| **过肩视角** | 读者看到的一切都经过主角的眼睛、耳朵、皮肤。极少宏观大场面 | 陈渡看不到的不写。陈渡感觉不到的不写。不是"第三人称限知"——是更激进的"摄像机在陈渡右肩上" |
| **规则驱动剧情** | 每道关卡是一道逻辑题。读者跟着主角一起想，不是被"告知"答案 | 陈渡破解规则时——读者必须能跟着他的思路走。每一步推理都要写出来，不能跳 |
| **原创关卡** | 所有副本独立设计，不套用现成IP | 每个规则异常有独立的逻辑。不是"又一个裂缝"——每次的裂缝逻辑不同 |
| **记忆点设计** | 标志性句式"我听到了XX的回响"伴随钟声与文字浮现，形成强烈的品牌化记忆 | 我们需要自己的"回响时刻"——一个反复出现、每次出现都让读者心跳的意象/句式 |
| **群像非模板** | 每个角色有独立小传和完整故事弧 | 配角不是信息站。配角有自己的"如果陈渡不来，他在做什么" |

### 警告：十日终焉的"纯文字"策略

学界评价它为"让文学回归文字"——利用文字的概括力和抽象性制造神秘感，读者必须放慢阅读速度。这要求**文字的精度极高**。如果句子不精确，慢下来反而暴露问题。

> **对我们来说**：白描能力是核心。写得不精确——十日终焉式的"慢读"就是灾难。写得精确——十日终焉式的"慢读"就是沉浸。

## 二、《异兽迷城》→ 取"日常厚度 + 不可逆代价"

### 为什么爆

- 番茄评分 9.7-9.8，抖音话题播放量超10亿，实体书六部
- 核心设定：99.99%的"人类"是兽伪装的——超级狼人杀
- 作者彭湃核心信条：**"都市脑洞文重要的不是脑洞，而是都市"**
- 主题：**"世界是假的，爱是真的"**

### 我们取什么

| 技法 | 说明 | 本作落地 |
|------|------|----------|
| **日常感的厚度** | 前期大量篇幅写高中校园——暗恋、考试、同学闲聊、二次元文化。这些细节符合读者的真实生活体验 | 陈渡的日常必须有可触的细节：长安北路堵车时的尾气味、出租屋热水器的点火声、楼下早餐摊的塑料袋。不是"十七平米"——是"十七平米里有什么味道" |
| **不可逆的代价** | 核心角色真实死亡。不是假死——是真的没了。读者会"怕" | 我们目前所有关键角色都"还在"——妈妈在走廊能对话、妹妹在背面能写字、顾伯黄昏还能见。缺少一个真正让读者怕失去的人或东西 |
| **悲情不为虐而虐** | 死亡服务于主题——"世界是假的，爱是真的" | 我们的代价必须服务于"妹妹为什么要留在背面"这个核心谜题 |
| **中国式生活肌理** | 父母唠叨、校园文化、年轻人生活习惯（拍照打卡、发朋友圈） | 陈渡的世界不需要"奇幻感"——需要"真实感"。他是真实的，纸才是假的 |
| **反地图叙事** | 所有剧情集中在一座城市。不换地图——向内挖 | 安城。就安城。棉纺三厂、槐安巷、护城河、槐南街。让读者能画出安城的地图 |

### 警告：异兽迷城的翻车点

- 配角模板化——"青灵的形容词永远是冷冷的"
- 后期战力崩坏——大幻觉套小幻觉
- 强行煽情——角色死亡频率过高，"为刀而刀"
- 中后期注水——战斗冗长，煽情套路重复

> **对我们来说**：代价≠杀人。不可逆≠每章死人。白线爬一寸、毛孔闭合、一段记忆消失——这些"小不可逆"的累积比大规模死亡更高级。

## 三、《神秘复苏》→ 取"规则精密度 + 规则碰撞"

### 为什么爆

- 规则怪谈类型的标杆。被评价为"写鬼超神"
- 核心创新：**鬼不是怨灵——是代码。** 每条鬼有自己的触发条件、作用范围、失效机制
- 最精彩的不是"主角打败鬼"——是**"让两个鬼的规则互相卡死"**

### 我们取什么

| 技法 | 说明 | 本作落地 |
|------|------|----------|
| **规则三层结构** | 每条规则必须拆解为：触发条件 → 作用范围 → 失效机制 | 陈渡遇到的每条规则异常，章纲里必须写清楚这三层。不写=不画设计图就盖楼 |
| **规则碰撞/卡BUG** | 利用两条规则的逻辑互相制约——这才是"破解规则的人"的核心动作 | 陈渡的能力不是"打破规则"——是"让规则A撞规则B，在抵消的瞬间做规则管不到的事" |
| **灵异工程化** | 灵异物品有设计-制造-使用的完整逻辑链 | 四滴（冰/铅笔/血/墨）每一滴需要：触发条件（怎么用）、作用范围（管什么）、失效机制（什么情况下不能用） |
| **中式民俗符号** | 鬼邮局、老宅送葬、红木家具、八音盒、柴刀、纸人、鬼新娘——赋予传统符号克苏鲁级别的威胁度 | 我们的"棉纺厂""书店""硬抄本""铅笔""门锁"——都是中式日常物。用日常物承载规则，比用"异界神器"高级 |
| **恐怖圈禁** | 用浓雾/黑暗/隧道视野把读者视野"圈禁"起来。灵异小说是"掌心的牢笼" | 规则异常场景必须封闭感。不是物理封闭——是规则封闭。读者和陈渡都出不去，直到规则被破解 |

### 三层规则设计模板

```
规则名称：
├── 触发条件（什么动作/时间/状态激活这条规则）
├── 作用范围（管什么、管谁、管到哪里为止）
├── 失效机制（什么情况下规则不生效/可以被绕过）
└── 碰撞可能（这条规则和哪条已知规则互为矛盾？）
```

### 警告：神秘复苏的翻车点

- "写鬼超神，写人超鬼"——人物塑造单薄，女性角色降智，配角工具人化
- 后期因审查从灵异被迫转"修仙"，大纲崩塌
- 现实部分文笔被指"小白文水平"

> **对我们来说**：人必须和规则一样精密。神秘复苏的教训——规则写得好可以让读者原谅人物单薄，但"原谅"不是"喜欢"。读者不会为规则流泪——只会为人流泪。

## 四、三书共同规律

### 1. 主角的"认知优势"是核心爽点

| 书 | 主角凭什么不一样 |
|----|------------------|
| 十日终焉 | 齐夏比其他人更早看清游戏的本质规律 |
| 异兽迷城 | 高阳觉醒了"看到兽"的能力——他比其他人多知道一层真相 |
| 神秘复苏 | 杨间驾驭了鬼——他能看到普通人看不到的规则 |

**我们的陈渡**：右手食指上的口子——能感知规则的裂缝。这就是他的认知优势。但认知优势不是"全知"——是"比别人多一个维度"。他知道裂缝在哪，但不知道裂缝通向哪。

### 2. 每章有独立消费价值

三本书都可以"翻到任意一章开始读"——因为每章有自己的问题-解决循环。不是"前情提要→本章纯过渡→下章待续"。

### 3. 信息不对等是追更的唯一引擎

读者追更不是因为"不知道结局"——是因为"读者和主角知道的东西不一样"。

| 不对等类型 | 效果 |
|-----------|------|
| 读者 > 主角 | "他要发现了！他要发现了！"（期待型追更） |
| 读者 < 主角 | "他怎么知道的？！他还要做什么？"（好奇型追更） |
| 读者 = 主角但都被骗了 | "原来是这样？！那一开始那个……"（颠覆型追更） |

三种不对等交替使用。不能永远一种。

### 4. 都有一个"品牌句"

| 书 | 品牌句 |
|----|--------|
| 十日终焉 | "我听到了XX的回响" |
| 异兽迷城 | "世界是假的，爱是真的" |
| 神秘复苏 | "能对付鬼的只有鬼" |

**我们缺这个。** 这条品牌句不是标语——是读者看完后向朋友推荐时会说的话。它必须同时概括世界观 + 主角状态 + 故事核心张力。

候选方向：
- "每条规则都有一个后门。"
- "纸上的字可以改。代价写在手上。"
- "她不是消失了。她是走到纸背面去了。"

## 五、差距清单（按优先级）

### 第一档：不改没法写

| # | 差距 | 行动 |
|---|------|------|
| 1 | **规则精密度** | 每条规则异常必须拆三层（触发条件/作用范围/失效机制）。写入章纲模板 |
| 2 | **规则碰撞** | 陈渡"破解规则"的核心动作必须是"让规则A撞规则B"。不是"找到裂缝钻过去"——而是"制造裂缝" |

### 第二档：前五章必须建立

| # | 差距 | 行动 |
|---|------|------|
| 3 | **品牌记忆点** | 找一句反复出现的意象/句式。每次出现都让读者心跳加速 |
| 4 | **日常感厚度** | 陈渡的日常必须有可触的细节——味道、声音、触感、习惯动作 |
| 5 | **不可逆代价** | 前五章内让读者"怕"一次——怕失去一个具体的人或东西 |

### 第三档：发展中自然积累

| # | 差距 | 行动 |
|---|------|------|
| 6 | **配角独立叙事** | 不急着给配角POV——但每个配角必须有"如果陈渡不来，他在做什么" |
| 7 | **本土符号系统** | 中式日常物承载规则——棉纺厂、书店、硬抄本、铅笔，不引入"异界神器" |
| 8 | **信息不对等交替** | 章纲的"信息涟漪"维度已经覆盖，持续关注即可 |
