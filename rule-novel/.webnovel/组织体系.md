# 组织体系

> 六个组织 + 背面生态。归零后重建。

---

## 一、守纸会

### 基本信息

| 项目 | 内容 |
|------|------|
| 创立者 | 周素云(奶奶) + 第一代裂缝发现者，约五十年前 |
| 总部 | 安城。具体位置只有会员知道 |
| 规模 | 约200-300人，全国分布，安城最多 |
| 标志 | 折纸——折四下。不是符号，是动作。守纸会的人见面不打招呼——互相折一张纸 |

### 核心信念

1. **纸很脆弱**——每一道裂缝都是纸在流血
2. **修，不要破**——规则不完美，但规则是纸的骨架。拆规则=拆纸
3. **代价=责任**——口子不是惩罚。口子=被纸选中。能力越大，修得越多
4. **正面优先**——保护正面的秩序。背面是"不得已的牺牲"

### 内部结构

```
会长(陆铮)
├── 长老会(孟婆+两人)
├── 外勤组(方如)——巡查裂缝、封补、应急处置
├── 档案室(温叔)——三百年裂缝记录、规则文献、无字书
├── 封补队(乔师傅)——专业封补裂缝
└── 学徒系统(小柯、双喜)——培养下一代
```

### 内部张力

- **陆铮 vs 孟婆**：陆铮想用陈渡——孟婆不想。她说"上次有人像他一样——是你奶奶。你奶奶走进去了，没出来。"
- **方如 vs 乔师傅**：方如修裂缝用新方法——乔师傅用老方法。新方法更快，但老方法更稳。方如修的一道裂缝后来裂了。乔师傅没说话——只是替她重新封了
- **阿七的叛逃**：三年前阿七从守纸会叛逃到拆纸会。不是理念认同——是他看到了守纸会档案室深处的某份文件。那份文件说了一件事：有些裂缝，守纸会故意不修。因为修了会暴露更大的裂缝
- **学徒断层**：老一辈在凋零。新一代口子出现的人越来越少。双喜是安城最后一个未成年人。如果他不来——守纸会三十年后没人了

### 对陈渡的态度

复杂。他是奶奶的孙子——自带遗产和期待。但他的做法（主动找裂缝、钻规则空子、让规则互相撞）和守纸会的理念（修裂缝、维持稳定、不要碰规则）完全相反。

陆铮想要他——但不知道能不能控制他。
孟婆不想碰他——但知道自己欠奶奶一句话。
方如观察他——想知道"不用规则修裂缝"的方法到底对不对。

## 二、拆纸会

### 基本信息

| 项目 | 内容 |
|------|------|
| 创立者 | 第一代背面归来者+体制受害者，约十五年前 |
| 总部 | 不固定。安城据点在旧工业区地下 |
| 规模 | 约100-150人，比守纸会小，但增长快 |
| 标志 | 撕裂的纸——不是折，是撕。竖着的裂缝 |

### 核心信念

1. **纸是牢笼**——物理定律是锁链。规则是看守
2. **砸碎它**——不管纸下面是什么。不可能比纸更糟
3. **代价=暴力**——口子不是礼物。是武器。用你的口子去打纸
4. **正面是谎言**——背面才是真相。被涂掉的人没有消失——他们在夹层里。把夹层打开

### 内部结构

```
安城头目(许墨)
├── 行动队(乌鸦)——破坏裂缝封补、爆破异常点
├── 理论部(季老师)——研究纸的本质、书写者的动机
├── 爆破组(小刀)——专门炸裂缝
├── 运输(柳)——在裂缝之间搬运"东西"
└── 温和派(沈)——不拆纸，主张掀开一角
```

### 内部张力

- **许墨 vs 沈**：许墨说"全拆"。沈说"先看"。两个人去年还一起行动——今年已经分头。但许墨没把沈踢出去。因为"拆纸会里需要有人说'等一下'——不然我不知道我是不是对的"
- **小刀的失控风险**：十五岁觉醒，父母被抹掉。只想烧。许墨喜欢她的纯度。乌鸦觉得她是炸弹——不是炸纸的炸弹，是炸自己人的炸弹
- **季老师的悖论**：不会任何规则能力——但他理解最深。拆纸会需要他写理论。但他的理论开始质疑拆纸本身的合理性。"纸是假的——但假的东西上面长了真的文明。你把纸拆了，文明还在吗？"
- **柳运的货**：没人问。没人敢问。有一次乌鸦瞥了一眼——货在动。不是活的——是"记得自己曾是活的"

### 对陈渡的态度

许墨第一眼看到陈渡的锁眼：**"终于。"**

不是"终于有盟友了"。是"终于有人能开那扇门了"。拆纸会炸了十年——炸不开大裂缝。锁眼是钥匙。陈渡是钥匙。

但沈觉得不对。"如果他开了门——然后呢？拆纸会想过'然后'吗？"

## 三、裂缝行者

### 基本信息

| 项目 | 内容 |
|------|------|
| 起源 | 自然形成。规则出裂缝→裂缝里有东西→有人靠裂缝吃饭→行者 |
| 总部 | 没有。裂缝在哪里，行者在哪 |
| 规模 | 无法统计。行者不登记 |
| 标志 | 没有。但行者能从一个人的步态判断他是不是行者——"走路的时候脚底留了缝" |

### 核心信念

1. **不站队**——守纸会要修裂缝，拆纸会要炸裂缝。行者不修也不炸。用裂缝
2. **裂缝是资源**——情报、入口、通道、交易品。裂缝里什么都有——包括"不应该出现在正面"的东西
3. **代价=运费**——用口子过裂缝，白线爬=付过路费。公平交易
4. **信息最值钱**——钟姐的情报比两家加起来都准。但行者不卖情报给任何一边——只卖情报给个人

### 内部结构

没有。行者自发分布。钟姐和瞎子刘算"前辈"——但不发号施令。双面是"中间人"——两边都打交道。胖头是"摆渡人"——收费带人。

有一种非正式的行者伦理：
- 不举报另一个行者（不管他给哪边干活）
- 裂缝的位置不白给（自己找，或者买）
- 不堵裂缝（行者不修裂缝——那是守纸会的事）
- 在裂缝里遇到了——不论之前在外面有什么恩怨——先一起出去

### 内部张力

- **瞎子刘 vs 新一代**：瞎子刘认为行者有底线——不碰"太深"的裂缝。老鼠在裂缝里长大——对老鼠来说每一道裂缝都是家。深浅没区别
- **钟姐的底线**：不卖"人"的情报。只卖裂缝的情报。有人问过她陈渡值多少——她说"不卖。出去。"
- **姜北的赌性**：不是所有的行者都赌。姜北赌。"从这条裂缝能出来吗？试试。出不来——那就不出来了。"老派行者不赌。姜北觉得他们怕了

### 对陈渡的态度

行者对陈渡的兴趣纯粹是商业性质的——他有锁眼。锁眼能开的裂缝比口子多。钟姐想和陈渡做交易。姜北想和陈渡赌一把。瞎子刘不想碰——"锁眼不是行者的东西。太深了。太深的裂缝——进去的不是行者，是不打算出来的人。"

## 四、管理局

### 基本信息

| 项目 | 内容 |
|------|------|
| 全称 | 安城特殊事件管理局 |
| 级别 | 市级。上级对口单位存在但不透明 |
| 规模 | 约60-80人。安城办公室 |
| 公开定位 | "处理无法用常规科学解释的社会安全事件" |

### 核心信念

1. **科学解释不了的——先记录**——二十年，两千多份档案。大部分没有结论
2. **维持秩序**——保护普通人。让他们不知道。必要的隐瞒=保护
3. **用规则对付规则**——老魏的刑侦方法论：异常事件有模式。分析模式→预测→预防

### 内部结构

```
局长(何局)
├── 行动科(唐杉)——现场处置、应急响应
├── 档案室(小秦)——记录、分析、建模
├── 技术组(小周)——电磁监测、信号分析
├── 后勤科(丁主任)——器材、场地、钱
└── 顾问(老魏)——退休返聘，不坐班
```

### 内部张力

- **唐杉 vs 老魏**：唐杉想"管"——主动出击，把异常关起来。老魏说"你管不了——你连'它'是什么都不知道"。唐杉不服——他在调查老魏的旧案
- **何局的秘密**：档案最深处锁了一份文件——"纸"。不是代号。文件本身是纸做的——但纸上的字是活的。每一任局长退休前会告诉下一任：文件不能烧。因为它不怕火。文件不能锁。因为锁不住。把文件放着——等一个知道它是什么的人来取
- **小秦的模型**：她用数据预测裂缝——准确率在上升。但如果她用模型找到了裂缝——管理局就有能力干预裂缝。干预裂缝=踏入守纸会和拆纸会的战场。何局还没做决定

### 对陈渡的态度

老魏是第一个注意到陈渡的官方人员——四十七起异常事件里，有三起"有目击者描述了一个手指有伤口的年轻男性"。他没上报。他在等——"那个人会自己走到管理局门口。"

唐杉也在等——但他想的是"按倒、控制、讯问"。

## 五、被涂者

### 基本信息

| 项目 | 内容 |
|------|------|
| 定义 | 被规则抹掉的人。规则判定他们"不属于正面"——从正面的记录里删除。但人没消失。卡在夹层 |
| 涂掉原因 | 规则不允许的行为 / 说了不该说的话 / 规则写错 / 被另一个人用血"交换" |
| 状态 | 不是活的不是死的。在夹层里存在。正面的普通人看不见他们。口子持有者可以 |

### 内部

没有组织。没有领袖。夹层里琴姨的"客栈"是最接近"据点"的地方——但也不是组织。被涂者来、坐、讲一个故事、走。有人再也没回来——可能走出去了，可能走进更深的夹层里去了。

### 涂掉的权利层级

被涂掉不是平等的。涂掉有"深浅"：

| 层级 | 状态 | 例子 |
|------|------|------|
| 浅层涂掉 | 在夹层浅处。规则的"删除"不彻底。黄昏时正面的人可以看到残影 | 顾伯 |
| 中层涂掉 | 完全在夹层里。正面不可见。但保留全部记忆和人格 | 程、林远、赵医生、诗人、琴姨 |
| 深层涂掉 | 规则的"删除"涉及本质——不只是"存在"，是"定义"。记忆开始模糊。人格开始变 | 告密者——他听到规则在念他的"罪名" |
| 极深涂掉 | 人不再是"人"——变成夹层的一部分。纤维的一部分 | ？|

### 内部张力

- **想回去的 vs 不想回去的**：赵医生想回去——还有病人没看完。赌徒不想——"在这里面我才是老大。外面规则太多，玩不开。"
- **记得正面的 vs 忘了正面的**：小女孩不知道"正面"是什么——她只在正面待了九年，在夹层等了二十年。她对正面的唯一记忆是"阳光。很暖。"
- **灰色人物**：告密者是被规则自己涂掉的——不是任何人操作的。规则判断他"危害了纸的完整性"。这是什么意思？规则有判断力？还是有人在背后操作规则？

## 六、书写者教派

### 基本信息

| 项目 | 内容 |
|------|------|
| 创立者 | 使徒。约十年前第一次出现 |
| 总部 | 不公开。传闻在安城某处桥附近 |
| 规模 | 小——可能不到50人。但极其投入 |
| 标志 | 缺了食指的右手。入教者不一定有口子——但入教后不能碰规则。"只有书写者可以写规则。你碰规则——就是僭越" |

### 核心信念

1. **书写者是唯一的书写者**——创造世界的那一位。不是神——是作者。世界是他的手稿
2. **规则不可触碰**——每一条规则都是书写者的笔迹。改规则=涂改他的手稿=亵渎
3. **裂缝是书写者在擦字**——不是故障。是他不满意。他在修改。不要干扰他的修改
4. **凌晨三点零七分是神圣的时间**——他在那个时间写字。全世界都应该在那个时间静默
5. **锁眼是反基督**——口子是纸给的能力（守纸会说是被选中，行者说是工具）。锁眼不是。锁眼是"打开书写者的稿子看底稿"的能力。亵渎中的亵渎

### 内部结构

```
使徒(精神领袖、最终解释者)
├── 沐(教义阐释、传教)
├── 哑巴(执行、保护)
└── 抄写员(记录规则、抄写经文)
```

### 内部张力

- **沐的过去**：曾是守纸会成员。见过一次规则在凌晨三点零七分自己重写了一行——没人碰。没人操作。纸自己改了自己的规则。守纸会说这是"纸的生长"。沐说不是——是"书写者在修改"。她离开守纸会。带走了一份档案。那份档案说"凌晨三点零七分，安城某处出现规则自修改现象。原因不明。建议继续观测。"她把档案烧了。"不能观测神。"
- **哑巴的选择**：不是被强迫沉默——是自己割了舌头。"我怕有一天我会说出'可能书写者不存在'。那时候——我已经没有舌头了。"
- **使徒的食指**：右手缺了食指。不是规则弄的——是自己弄的。"书写者用食指写世界。我不配。"但有人注意过——他的右手食指断面有愈合的痕迹。愈合方式很像口子。不是他割了食指——是他的口子长在食指上，然后他把整根食指去掉了

### 对陈渡的态度

**敌人。** 锁眼是最大的亵渎。使徒第一次听说陈渡的锁眼——据说沉默了很久。然后说了一句话："他不是敌人。他是考验。书写者在用他考验我们。"

沐不同意。"他不是考验。他是书写者写错的一个字。应该被擦掉。"

## 七、背面生态

### 不是组织——是生态

背面不是管理出来的。是自然生长的。规则和裂缝在背面摩擦——产生了自己的秩序。

### 地理概念

| 区域 | 描述 |
|------|------|
| **浅背** | 最靠近正面的区域。纸薄。可以透过纤维看到正面的光。陈屿修裂缝的主要区域。老兵的活动范围 |
| **中背** | 纤维层。规则在这里以"反写"的形式存在。守门人站在这个区域的中心 |
| **深背** | 纤维底层。掘墓人在这里挖。传说——如果穿透底纤维——会看到"纸之前的东西" |
| **大裂缝** | 从中背延伸到深背的一道裂缝。在扩大。守门人站在裂缝边。妹妹每天用铅笔在裂缝边缘写 |
| **桥** | 书写者坐的地方。不在正面——不在背面——在纸的第一个字。那个字本身就是一座桥 |

### 规则的不同表现

正面的规则是"必须遵守"。背面的规则是"可以看见但碰不到"——像写在水里的字。看得见。捞不起来。

在背面待久了——规则开始对你不一样。不是失效——是"认识"你。绘图师说她画的纤维地图每天在变——不是因为纤维在变。是纸在躲她。"它不想被画下来。它害羞。"

### 和六组织的关系

| 组织 | 和背面的关系 |
|------|-------------|
| 守纸会 | 认识背面。封补裂缝就是"不让背面的东西漏到正面"。但不想任何人长期待在背面——包括陈屿。觉得"她应该回来" |
| 拆纸会 | 想打开背面。不是进背面——是把正面砸了，让背面取代正面。许墨进过背面——回来以后变了。他看见的东西可能是背面被他隐瞒了 |
| 裂缝行者 | 走裂缝的知道怎么从裂缝"瞥"一眼背面。钟姐进去过。瞎子刘说"背面不是你去的地方。你只从裂缝的边缘蹭过去——就够了" |
| 管理局 | 理论上不知道。但何局的那份"纸"文件——在凌晨三点零七分——有时会从纸页的背面浮出字来。不是正面的字。是背面的。小秦没注意到——她在看正面的档案 |
| 被涂者 | 住在夹层——不是背面。但夹层和背面之间的界限越来越薄。琴姨说"以前夹层是夹层，背面是背面。现在不行了。裂缝大了。夹层和背面之间——在漏" |
| 书写者教派 | 认为背面是"书写者的草稿纸"。不应该有任何人踩在他写废的稿子上。"沐说背面不是世界——是垃圾桶。书写者写坏了，放在背面。妹妹在翻他的垃圾桶。" |
