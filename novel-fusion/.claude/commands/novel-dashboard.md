# /novel:dashboard — 可视化面板

打开项目状态面板，一站式查看所有关键信息。

## 面板内容

### 📊 进度概览
- 总章节数 / 已写完 / 已规划
- 总字数
- 当前卷进度
- 最近更新时间

### 👥 角色面板
- 主角当前状态（位置/等级/情绪）
- 主要配角状态一览
- 反派动态

### 🎯 伏笔面板
- 开放伏笔数量
- 即将到期（计划回收在 5 章内）
- 已过期未回收（⚠️ 需要处理）
- 伏笔回收率

### 📈 质量趋势
- 近 10 章评分曲线
- 各维度平均分雷达图
- 问题频率统计

### 🧵 故事线面板
- Strand 三线进度条
- 各线最近更新章节
- 断档警告

### 🏆 爽点面板
- 8 大模式使用频率
- 最近一次里程碑胜利
- 爽点密度趋势

### 💾 系统状态
- Story System 健康度
- 投影同步状态
- 上次备份时间

## 使用示例
```
/novel:dashboard          # 完整面板
/novel:dashboard 伏笔     # 只看伏笔面板
/novel:dashboard 角色     # 只看角色面板
/novel:dashboard 质量     # 只看质量趋势
```
